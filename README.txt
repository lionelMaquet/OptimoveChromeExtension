Cette extension Chrome permet d'agrandir les fenêtres Optimove. 

Pour installer cette extension : 

1) Téléchargez ce dossier via le bouton Code -> Download Zip
2) Dézippez le dossier sur votre PC
3) Allez à l'adresse chrome://extensions/
4) Cochez "Developer Mode" en haut à droite de l'écran 
5) Cliquez sur le bouton "Load Unpacked" en haut à gauche de l'écran
6) Sélectionnez le dossier que vous venez de dézipper

Tout devrait fonctionner.
Si ça ne fonctionne pas, essayez de relancer votre navigateur. 

Si vous souhaitez désactiver cette extension : 

1) Allez à l'adresse chrome://extensions/
2) En bas à droite du cadre "Optimove Templates CSS Modifier", décochez l'extension
